# Function introduction (some commonly used ones are listed, the rest are in the file)
==基础运算:“+”、“-”、“*”、“/”、“%”==
计算器扩展的“+”、“-”、“*”、“/”可以实现分数计算，如“1/2+1/3”，结果就是“5/6”，但“分数+小数”的结果类型是小数，后续版本可能会改
“%”暂时不能弄分数、小数，如：“10%3”，结果就是“1”
==比较运算:“>”、“<”、“=”、“≡”、“≤”、“≥”==
“>”现在也可以弄分数，如“1/2>0.3”，结果是“true”
“<”照样
“=”最常见，如“(1/2)=0.5”，结果是“true”
“≡”差不多会用到一点点，如“1/2=0.5”，结果是“false”
“≤”用到的比较多，如“0.5≤1/2”，结果是“true”
“≥”也一样，如“1/2≥0.5”，结果是“true”
