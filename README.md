# Function introduction (some commonly used ones are listed, the rest are in the file) (功能介绍（列出了一些常用的，其他的在文件中）)
==基础运算:“+”、“-”、“*”、“/”、“%”==

计算器扩展的“+”、“-”、“*”、“/”可以实现分数计算，如“1/2+1/3”，结果就是“5/6”，但“分数+小数”的结果类型是小数，后续版本可能会改
“%”暂时不能弄分数、小数，如：“10%3”，结果就是“1”

==比较运算:“>”、“<”、“=”、“≡”、“≤”、“≥”==

“>”现在也可以弄分数，如“1/2>0.3”，结果是“true”
“<”照样
“=”最常见，如“(1/2)=0.5”，结果是“true”
“≡”差不多会用到一点点，如“1/2=0.5”，结果是“false”
“≤”用到的比较多，如“0.5≤1/2”，结果是“true”
“≥”也一样，如“1/2≥0.5”，结果是“true”

==Basic operations: " ", "-", "*", "/", "%"==

The calculator's extended " ", "-", "*", "/" can perform fraction calculations, such as "1/2 1/3", resulting in "5/6", but the result type of "fraction decimal" is a decimal; this may be changed in future versions.
The "%" operator currently cannot handle fractions or decimals, for example: "10%3" results in "1".

==Comparison operations: ">", "<", "=", "≡", "≤", "≥"==

">" can now also handle fractions, for example "1/2>0.3" results in "true".
"<" works similarly.
"=" is the most common, for example "(1/2)=0.5" results in "true".
"≡" is used a little bit, for example "1/2=0.5" results in "false".
"≤" is used quite often, for example "0.5≤1/2" results in "true".
"≥" is similar, for example "1/2≥0.5" results in "true".

The rest will not be mentioned here. After importing, click to open the document, and you can understand it yourself.
