v1.0.2

*发布时间：2026-02-24*

这版本加了新东西：

🚀 重磅新功能：复数运算

复数加减乘除都能算：2+3i + 1+2i = 3+5i

共轭复数：2+3i的共轭是2-3i

复数模长：3+4i的模是5

格式支持：2+3i、-1-2i、5i、-3i、纯实数也行

📊 矩阵运算来了

矩阵加法：[1,2;3,4] + [2,3;4,5] = [3,5;7,9]

矩阵乘法：支持2x2和3x3矩阵相乘

矩阵数乘：整个矩阵乘以一个数

矩阵转置：行变列，列变行

行列式计算：2x2和3x3矩阵都能算

🎲 概率统计

二项分布概率：n次试验成功k次的概率

正态分布概率密度：给定μ和σ，算某个x的概率密度

期望值计算：给一组值和对应的概率，算期望

概率和必须为1，不是1会报错

📏 单位转换

长度转换：米、厘米、毫米、千米、英寸、英尺、英里

温度转换：摄氏度、华氏度、开尔文

想转啥转啥，1米=100厘米，100华氏度≈37.78摄氏度

🧮 方程求解

一元一次方程：ax + b = 0，解出x

一元二次方程：ax² + bx + c = 0，支持实数根和复数根

判别式大于0两个实根，等于0一个重根，小于0两个复根

📈 微积分基础

求导数：f(x)=x²在x=2的导数是4

求积分：∫x²dx从0到1 ≈ 0.333

支持函数：x、x²、x³、√x、1/x、sin(x)、cos(x)、e^x、ln(x)

✨ 特殊数学函数

Γ函数：Γ(5)=24（就是4!）

误差函数erf：统计里常用的那个

贝塞尔函数：J₀和J₁阶的近似计算

➕ 其他新增

进制转换支持2-36进制了，以前只有二进制

增加了黄金比例φ ≈ 1.618

几何运算加了圆面积、圆周长、三角形面积

统计运算加了方差

比较运算终于来了：>、<、=、≡、≤、≥

🔧 优化和改进

Version 1.0.2 

*Release Date: February 24, 2026* 

This version has added new elements: 

🚀 Major New Feature: Complex Number Operations 

Addition, subtraction, multiplication and division of complex numbers can all be calculated: 2 + 3i + 1 + 2i = 3 + 5i 

The conjugate of a complex number: The conjugate of 2 + 3i is 2 - 3i. 

The modulus of the complex number 3 + 4i is 5. 

2 + 3i, -1 - 2i, 5i, -3i, pure real numbers are also acceptable. 

📊 Matrix operations are here! 

Matrix addition: [1, 2; 3, 4] + [2, 3; 4, 5] = [3, 5; 7, 9] 

Matrix multiplication: Supports multiplication of 2x2 and 3x3 matrices. 

Scalar multiplication of a matrix: Multiplying the entire matrix by a number. 

Matrix transposition: rows become columns, columns become rows. 

Determinant calculation: Can calculate both 2x2 and 3x3 matrices. 

🎲 Probability and Statistics 

Binomial distribution probability: The probability of k successes in n trials. 

Normal distribution probability density: Given μ and σ, calculate the probability density of a certain x. 

Expectation calculation: Given a set of values and their corresponding probabilities, calculate the expectation. 

The sum of probabilities must be 1. If it is not 1, an error will be reported. 

📏 Unit Conversion 

Length conversion: meters, centimeters, millimeters, kilometers, inches, feet, miles 

Temperature Conversion: Celsius, Fahrenheit, Kelvin 

You can convert whatever you want. 1 meter equals 100 centimeters, and 100 degrees Fahrenheit is approximately 37.78 degrees Celsius. 

🧮 Equation Solving 

Linear equation in one variable: ax + b = 0, solve for x 

Quadratic equation: ax² + bx + c = 0, supporting real roots and complex roots 

If the discriminant is greater than 0, there are two real roots; if it is equal to 0, there is one repeated root; if it is less than 0, there are two complex roots. 

📈 Fundamentals of Calculus 

To find the derivative: The derivative of f(x) = x² at x = 2 is 4. 

The integral of ∫x²dx from 0 to 1 is approximately 0.333. 

Supporting functions: x, x², x³, √x, 1/x, sin(x), cos(x), e^x, ln(x) 

✨ Special Mathematical Functions 

Gamma function: Γ(5) = 24 (which is 4!) 

The error function erf: the one commonly used in statistics. 

Approximate Calculation of Bessel Functions: Orders J₀ and J₁ 

➕ Other new additions 

Base conversion now supports 2 to 36 bases, while previously only binary was available. 

The golden ratio φ ≈ 1.618 has been added. 

Geometric operations have been added for the area of a circle, the circumference of a circle, and the area of a triangle. 

The statistical operation has added variance. 

Comparison operations are finally here: >, <, =, ≡, ≤, ≥ 

Optimization and improvement 

The editor's detection is more accurate now, especially for Turbowarp. 

Fixed some bugs in fraction calculation.
