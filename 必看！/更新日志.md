v1.0.2
*发布时间：2026-02-24*

这版本加了好多新东西，我自己都数不清了：

🚀 重磅新功能：复数运算

复数加减乘除都能算：2+3i + 1+2i = 3+5i

共轭复数：2+3i的共轭是2-3i

复数模长：3+4i的模是5

格式支持：2+3i、-1-2i、5i、-3i、纯实数也行

📊 矩阵运算来了

矩阵加法：[1,2;3,4] + [2,3;4,5] = [3,5;7,9]

矩阵乘法：支持2x2和3x3矩阵相乘

矩阵数乘：整个矩阵乘以一个数

矩阵转置：行变列，列变行

行列式计算：2x2和3x3矩阵都能算

🎲 概率统计

二项分布概率：n次试验成功k次的概率

正态分布概率密度：给定μ和σ，算某个x的概率密度

期望值计算：给一组值和对应的概率，算期望

概率和必须为1，不是1会报错

📏 单位转换

长度转换：米、厘米、毫米、千米、英寸、英尺、英里

温度转换：摄氏度、华氏度、开尔文

想转啥转啥，1米=100厘米，100华氏度≈37.78摄氏度

🧮 方程求解

一元一次方程：ax + b = 0，解出x

一元二次方程：ax² + bx + c = 0，支持实数根和复数根

判别式大于0两个实根，等于0一个重根，小于0两个复根

📈 微积分基础

求导数：f(x)=x²在x=2的导数是4

求积分：∫x²dx从0到1 ≈ 0.333

支持函数：x、x²、x³、√x、1/x、sin(x)、cos(x)、e^x、ln(x)

✨ 特殊数学函数

Γ函数：Γ(5)=24（就是4!）

误差函数erf：统计里常用的那个

贝塞尔函数：J₀和J₁阶的近似计算

➕ 其他新增

进制转换支持2-36进制了，以前只有二进制

增加了黄金比例φ ≈ 1.618

几何运算加了圆面积、圆周长、三角形面积

统计运算加了方差

比较运算终于来了：>、<、=、≡、≤、≥

🔧 优化和改进

编辑器检测更准确了，特别是Turbowarp

文档内容大更新，把所有新功能都写进去了

修复了一些分数计算的bug

Linear equations: ax + b = 0 → solve for x
